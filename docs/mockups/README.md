# 医療カルテシステム 画面モック

## 概要

このディレクトリには、医療カルテシステムの主要画面のHTMLモックが含まれています。

## モック一覧

### 🔐 認証・共通画面
- **login.html** - ログイン画面
- **dashboard.html** - ダッシュボード（ホーム画面）

### 👥 患者管理画面
- **patient-search.html** - 患者検索画面

### 📝 診療管理画面
- **medical-record.html** - カルテ記録画面（SOAP形式）

### 💊 オーダー管理画面
- **prescription-order.html** - 処方オーダー入力画面

## 使い方

1. **ブラウザで開く**
   ```
   docs/mockups/index.html をブラウザで開く
   ```

2. **または、個別の画面を直接開く**
   ```
   docs/mockups/login.html から開始
   ```

3. **画面遷移**
   - ログイン → ダッシュボード → 各機能画面へ遷移可能
   - ナビゲーションバーとサイドバーで画面を切り替え

1. **レスポンシブデザイン**
   - PC、タブレット対応

   - ログイン → ダッシュボード

## 操作ガイド

### 1. ログイン画面
- 任意のユーザーID・パスワードを入力
- 「ログイン」ボタンでダッシュボードへ

### 2. ダッシュボード
- 本日の予定、アラート、統計を表示
- クイックアクセスボタンから各機能へ
- 予約患者一覧から診察開始

### 3. 患者検索
- 検索条件を入力（患者番号、氏名カナ、生年月日等）
- 検索結果から患者を選択
- 「詳細」ボタンまたは行クリックで患者情報へ

### 4. カルテ記録
- 患者選択後、SOAP形式で記録
  - S: 自覚症状（患者の訴え）
  - O: 他覚所見（検査結果等）
  - A: 評価（診断）
  - P: 計画（治療方針）
- 「一時保存」で下書き保存
- 「署名して確定」でカルテ確定

### 5. 処方オーダー
- 薬剤を検索・選択
- 用法・用量を入力
- アレルギーチェック結果を確認
- 警告が出た場合は確認ダイアログで判断
- 「オーダー」ボタンで処方確定
